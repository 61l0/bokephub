Changing the default Icons - WP Socializer

Default Icons in this folder can be replaced with custom WP Socializer icons. To replace the default icons please note these steps

  • Replace the icons with the same name of the default.
  • Replace with only 32px Icons.

Otherwise you can choose another folder containing the custom 32px Icons with the same name. The path of the new folder should be changed in the admin page.
